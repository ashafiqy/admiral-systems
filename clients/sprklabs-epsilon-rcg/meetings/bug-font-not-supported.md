# Summary

Valentine found a new BUG on the RCG staging website.

The font "Ramabhadra, sans-serif" does not support the TM (trademark symbol) -> ™

## IMPORTANT CONTEXT FOR THE TEAM

It is super important for the team to know, that Scott SPECIFICALLY SAID he wants all the frameworks to have the TM symbol after the framework name. It MUST be both inside the CMS title and also the Framework PDF file

## Potential fixes

Valentine suggested using a TM image instead if we can't find a fix to the font via Webflow.

## Side information on the TM symbol

To type the TM symbol:
Press and hold ALT+0153 for the tm symbol ™ or ALT+0169 for the copyright symbol c ©.
